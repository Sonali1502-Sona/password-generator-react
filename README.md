## 🚀 Local Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```

> Hot reload is enabled automatically via Vite.

## ✨ Features

- 🎨 Glassmorphism dark UI with animated ambient blobs
- 🔢 Adjustable password length (6–32 characters) via slider
- 🔡 Toggle numbers and symbols independently
- 💪 Live strength indicator — Weak / Fair / Good / Strong / Very Strong
- 📋 One-click copy with visual confirmation
- ♻️ Auto-regenerates on every setting change
